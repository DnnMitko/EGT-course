Tetris clone written in SDL

Controls:
up - rotate clockwise 90 degrees
left, right - move shape one block to the side
down - speed up drop
space - slam the shape down

Implemented so far:
- Complete basic game mechanics (rotation, move, colision checks, line completion)
- Shape generation
- Show next shape
- Show and calculate score
- Speed up shape drop
- Slam shape down
- Hold shape
- Menu
